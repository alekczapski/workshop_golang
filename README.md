## Topics

Formats: 2 days, 1 day, 4h

### Basics

- packaging
- go get, go build, go install
- fmt, imports, linter
- vars, consts
- functions
- data types
- for, if, switch
- pointers
- structs
- arrays and slices
- maps
- methods

### Mid-level

- interfaces
- goroutines
- channels
- error handling
- (file) io
- unit tests

### Advanced (not really)

- stdlib
- ide: AC, def/usage, debug
- package/dependency management
- marshall/unmarshall (sed/deser)
- docs
- benchmarking
- calling remote apis
- parsing cli args
- db access
- logging
